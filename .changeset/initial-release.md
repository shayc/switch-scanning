Initial public release: a framework-agnostic switch-scanning engine with optional React bindings. Includes four established scan methods (automatic, two-switch step, single-switch dwell, and inverse), nested row–column groups with a guaranteed escape, stabilized physical input (tremor filtering, repeat suppression, tap/hold recognition, and stuck-switch cleanup), and imperative highlight movement that does not rerender the React tree.
